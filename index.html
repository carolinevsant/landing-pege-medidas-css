Unidades de medidas do CSS

Essenciais para criar layouts responsivos, permitindo definir tamanhos, espaçamentos e dimensões dos elementos de forma que as páginas se ajustem a diferentes dispositivos e tamanhos de tela.

Absolutas ⬇️

Representam valores fixos que não se adaptam ao tamanho da tela ou configurações do usuário. Elas são úteis quando se deseja um controle preciso das dimensões, como em impressões ou layouts estáticos

Relativas ⬇️

As unidades relativas em CSS são vantajosas porque se adaptam a diferentes tamanhos de tela, facilitam a criação de layouts responsivos e melhoram a acessibilidade, tornando os sites mais flexíveis e fáceis de manter.

Angulo ⬇️

Unidades de ângulo em CSS, como graus (deg), são usadas para rotacionar elementos em uma página. Por exemplo, é possível girar uma imagem ou um botão ao passar o mouse por cima.
